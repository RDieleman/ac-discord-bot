# AC Discord bot
This project was a discord bot used to manage the a Oldschool Runescape clan AC.
The bot provides the connection between the discord server and external services like the website, database and calendars.

It implements functions like:
- Handling the interviews with potential members.
- Tracking attendance for events in voice channels
- Providing a synched calendar
- Providing members with stats about their activity
- Deleting messages
